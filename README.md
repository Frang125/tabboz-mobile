Salve. Desideravo da anni realizzare un porting di Tabboz su mobile, ma non conoscendo il linguaggio C né le librerie specifiche per Android, avevo accantonato questo desiderio. Con la nascita della versione web ho avuto finalmente la possibilità di creare un porting tramite la WebView di Android. Detto in parole semplici, questa versione è un wrapper della versione web.

Note:

Il seguente porting presenta alcuni bug relativi ai dialoghi del gioco; per risolverli è necessario chiudere e riaprire l’app.

Il porting è ancora acerbo, poiché ho lavorato poco su questo progetto.

Purtroppo il gioco è stato testato solo sul mio tablet (Lenovo Idea Tab 2025), quindi non so se funzioni correttamente sui telefoni Android.

L’app occupa circa 11 MB perché include tutto il codice della versione web (scelta fatta per permetterne il funzionamento anche senza connessione a Internet).
